# King Vibez — Fullstack Starter (Complete)

This is a full-stack starter application for the King Vibez rapper website:
- Frontend: React (Vite) + simple Tailwind-like CSS
- Backend: Node.js + Express + MongoDB (Mongoose)
- Auth: JWT + bcrypt
- Media: placeholders for Cloudinary upload
- Payments: Stripe placeholders

Follow instructions in `backend/README.md` and `frontend/README.md` to run locally.
